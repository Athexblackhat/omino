// Installation

Deploy in Minutes

Native packages for Debian, RHEL, Arch, and macOS. Or go Docker for universal deployment across any environment.

bash — omino installer
# Clone the repository
$ git clone https://github.com/Athexblackhat/OMINO.git
Cloning into 'OMINO'...
Receiving objects: 100% — done.

$ cd OMINO

# Interactive installer (auto-detects your distro)
$ sudo bash install.sh

[*] Detected: Kali Linux (Debian-based)
[*] Installing dependencies...
[*] Configuring 90+ tools...
[✓] OMINO Black Hat Edition ready.
[✓] Developed by ATHEX BLACK HAT × DIR CYBER

# Force install (no prompts)
$ sudo bash install.sh force
docker — kali / blackarch
# Kali Linux base image
$ sudo docker compose up
$ sudo docker run --privileged -it OMINO-kali-linux /bin/bash

# BlackArch base image
$ sudo docker compose -f docker-compose-blackarch.yml up
$ sudo docker run --privileged -it OMINO-blackarch /bin/bash
bash — macOS + homebrew
# Install Homebrew first
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Clone and install
$ git clone https://github.com/Athexblackhat/OMINO.git && cd OMINO
$ sudo bash install.sh
